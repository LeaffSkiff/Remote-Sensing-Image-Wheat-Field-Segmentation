## 快速开始

### 训练模型
```bash
# 方法 1：使用批处理脚本
scripts\train.bat

# 方法 2：命令行
C:\Users\28027\miniconda3\envs\mmseg_env\python.exe ^
  C:\Users\28027\miniconda3\envs\mmseg_env\Lib\site-packages\mmseg\tools\train.py ^
  mmseg\configs\wheat\wheat_unet_r50.py ^
  --work-dir mmseg\work_dirs\wheat_unet
```

### 半自动标注
```bash
scripts\batch_workflow.bat
```

## 使用说明

1. **准备数据**：将图像放入 `to_predict/`，运行 `scripts/train.bat` 选择准备数据集
2. **开始训练**：运行 `scripts/train.bat` 选择训练模型
3. **预测**：将新图像放入 `to_predict/`，运行预测脚本
4. **微调标注**：使用 `scripts/batch_workflow.bat` 转 labelme 格式（输出到 `data_set/labelme_annotations/`），微调后导出

详细文档见 `docs/` 文件夹。
