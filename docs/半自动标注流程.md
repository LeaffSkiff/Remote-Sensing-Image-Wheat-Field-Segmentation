# 半自动标注工作流程

## 工作流程图

```
┌─────────────┐     ┌──────────────┐     ┌─────────────┐
│  原始图像   │ ──► │  模型预测    │ ──► │ 预测 mask   │
└─────────────┘     └──────────────┘     └─────────────┘
                                              │
                                              ▼
┌─────────────┐     ┌──────────────┐     ┌─────────────┐
│ 重新训练    │ ◄── │  新训练数据  │ ◄── │ labelme 微调 │ ◄─── 导出
└─────────────┘     └──────────────┘     └─────────────┘
```

## 步骤 1: 用现有模型预测

```bash
# 使用 conda 环境的 Python
C:\Users\28027\miniconda3\envs\mmseg_env\python.exe ^
  mmseg\tools\predict_mmseg.py ^
  --config mmseg\configs\wheat\wheat_unet_r50.py ^
  --checkpoint mmseg\work_dirs\wheat_unet\best.pth ^
  --image to_predict ^
  --output predictions
```

## 步骤 2: 转 labelme 格式（用于微调）

```bash
C:\Users\28027\miniconda3\envs\mmseg_env\python.exe ^
  mmseg\tools\pred_to_labelme.py ^
  --pred predictions ^
  --image to_predict ^
  --output data_set\labelme_annotations
```

## 步骤 3: 在 labelme 中微调

1. 打开 labelme
2. 打开 `data_set\labelme_annotations\` 中的 JSON 文件
3. 使用多边形工具编辑边界
4. 保存

## 步骤 4: 导出为训练数据

```bash
C:\Users\28027\miniconda3\envs\mmseg_env\python.exe ^
  mmseg\tools\labelme_to_mmseg.py ^
  --input data_set\labelme_annotations ^
  --img-out data/processed/images/train ^
  --mask-out data/processed/annotations/train
```

## 步骤 5: 重新训练

```bash
C:\Users\28027\miniconda3\envs\mmseg_env\python.exe ^
  C:\Users\28027\miniconda3\envs\mmseg_env\Lib\site-packages\mmseg\tools\train.py ^
  mmseg\configs\wheat\wheat_unet_r50.py ^
  --work-dir mmseg\work_dirs\wheat_unet_v2
```
